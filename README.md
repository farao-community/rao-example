# rao-example
An example of a Java app using Powsybl-OpenRAO.
The code described in the [tutorial](https://powsybl.readthedocs.io/projects/openrao/en/latest/getting-started/tutorial.html) can be found in the Main class.

# To run main in the command line

Create the jar: 

```
mvn clean package 
```

Run the main:

```
java -jar target/rao-example-1.0-SNAPSHOT.jar
```
